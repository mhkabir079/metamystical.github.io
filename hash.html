Hashing instructions

  1. A random set of 12 easy-to-remember English words is generated every time this page is refreshed.
  2. This mnemonic text is hashed unless other text is entered instead. Leading and trailing spaces are ignored in the text box.
  3. Use the Hash mnemonic button to hash the text. Only the first 256 bits (32 bytes) of the SHA-512 hash are displayed.
  4. The hashed mnemonic can be used as a secret to digitially sign or to encrypt.
